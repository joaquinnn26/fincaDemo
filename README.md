# Finca El Encinar

Landing page moderna para una finca boutique orientada a bodas, eventos, escapadas rurales y celebraciones privadas.

## Tecnologías
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`

## Estructura destacada
- `app/` para rutas y metadatos
- `components/` para bloques reutilizables
- `data/` para contenidos de demostración
- `public/images/` para fotografías de ejemplo

## Personalización
- Ajusta textos, datos y WhatsApp en `data/site.ts`
- Sustituye las imágenes de `public/images/`
- Conecta el formulario de presupuesto a un backend real en el futuro
